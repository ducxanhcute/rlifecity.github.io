Mitarbeiter Ansicht

Mitarbeiter erhalten Zugriff auf eine Desktopansicht, über welche sie als Admin administrativ tätig werden können oder als normaler Mitarbeiter Zugriff auf alle Informationen der jeweiligen Kunden erhalten können.

Kunden Ansicht

Nicht nur Mitarbeiter erhalten Zugriff auf das System. Kunden können über einen zuvor generierten Zugangscode jederzeit den Fortschritt Ihrer Lizenzen sowie ihren Ausgaben einsehen. Die Zugänge können auf Wünsch auch wieder gelöscht werden.

Passe das System an

Uns ist es wichtig, dass du das System so benutzen kannst wie es für deinen Server richtig ist. Deshalb können das Logo sowie Fragebögen und Lizenztypen auf Wünsch angepasst und so das System individualisiert werden.

Abrechnungssystem

Mitarbeiter können Spielern über unser Abrechnungssystem Rechnungen für die Theorie sowie Praxisprüfung ausstellen. Dafür können Produkte sowie Preise zuvor von einem Admin festgelegt werden.
